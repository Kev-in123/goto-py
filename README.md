# goto.py

This package allows you to go back to a certain line.

## Installation

Stable version:

```
pip install goto-py
```

Development version:

```
pip install git+https://github.com/Kev-in123/goto-py
```

Refer to [test.py](https://github.com/Kev-in123/goto-py/blob/main/test.py) for an example
